Los Fundamentos Basicos Del Lenguaje CSS


¿Que es CSS?

 Segun Jáimez, CSS es un lenguaje de marcado centrado en definir, crear y mejorar la representación de documentos basados ​​en HTML.

¿Para qué sirve el CSS?

Segun Jáimez, sirve El código CSS facilita la vida de los desarrolladores front-end al separar la estructura de un documento HTML de su presentación. En otras palabras, HTML actuará como el esqueleto de la Web, definiendo su estructura básica, mientras que CSS agregará toda la capa de personalización sobre la que la Web define su apariencia final.

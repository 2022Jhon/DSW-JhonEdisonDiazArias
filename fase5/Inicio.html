Propósito del sitio


El propósito de este sitio web es para informar a la población en general de los fundamentos principales que existen hasta el momento en el lenguaje CSS “Cascading Style Sheets” y los 5 mejores frameworks de CSS.


Introducción


Mediante la presente actividad se realiza con el fin de poner en practica todo lo aprendidos del lenguaje de CSS “Cascading Style Sheets” y colocando todo tipo de etiquetas para observar el sitio web mas atractivo al usuario final.
